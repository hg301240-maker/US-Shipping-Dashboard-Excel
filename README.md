# 🚚 US Shipping Dashboard - Excel

An interactive Microsoft Excel dashboard designed to analyze US shipping data and provide valuable business insights. This project helps users monitor sales performance, shipping efficiency, profit trends, and key business KPIs through dynamic visualizations and interactive filters.

---

## 📌 Project Overview

The US Shipping Dashboard is built using Microsoft Excel to transform raw shipping data into meaningful business insights. It enables users to quickly analyze sales, profit, orders, shipping performance, and regional trends using Pivot Tables, Charts, and Slicers.

This dashboard is suitable for business analysts, students, and anyone interested in learning Excel-based data analytics.
